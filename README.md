# LuckyStarter
 Webpack + Twig Boilerplate for frontend templating

# Features
- Twig Templating Engine
- Webpack + Browsersync
- Tailwind CSS with Purge Css for production
- More features are in development

# How to use
Start by cloning the project. Then run the command
``` npm install . ```

For Development
``` npm run watch ```

For Production
``` npm run production ```
